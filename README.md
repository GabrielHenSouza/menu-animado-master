# menu-animado-master

Pequeno projeto de menu animado em HTML/CSS/JS.

Arquivos principais
- `index.html` – página principal.
- `styles.css` – estilos.
- `index.js` – comportamento/JS.

Como usar
1. Abra `index.html` no seu navegador (duplo clique ou `File -> Open`).
2. Não há dependências externas; é um projeto estático simples.
